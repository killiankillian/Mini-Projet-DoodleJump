Projet Informatique : Doodle Jump

Description du projet 
=========
Dans le cadre du mini projet de 2ème année, nous avons réalisé un jeu du type Doodle Jump. Le jeu se contrôle à l’aide des flèches droite et gauche, elles permettent de déplacer le personnage qui doit sauter de plateformes en plateformes pour arriver le plus haut possible. Il faut prendre garde à ne pas tomber en dehors de l’écran et de ne pas toucher les plateformes rouges qui sont des ennemies. 

Installation 
=========
Pour utiliser notre jeu, il n’y a besoin que de Geany. 
Après avoir télécharger le fichier source, il faut modifier les chemins d’appel aux images pour qu’ils correspondent à l’endroit où vous avez mis le fichier sur votre ordinateur. Les lignes à modifier se trouvent dans les classes FenetreJeu, FenetreDemarrage, et FenetreMort.
La classe contenant le main est GestionFenetre.

Description des images utilisées:
=============
- fond.png: utiliser comme fond pour toutes nos fenêtres
- Doodle.png: personnage du jeu
- Titre.png: sert à la mise en forme de la FenetreDemarrage
- perdu.png: sert à la mise en forme de la FenetreMort
- palier.png: permettent un saut normal. Ce sont les paliers verts
- palier1.png: paliers bleus qui permettent un petit saut
- palier2.png: paliers gris qui entrainent un saut moyen
- palier3.png: paliers avec un ressort qui permettent un super saut
- palier4.png: paliers blancs qui disparaissent après un saut
- palier5.png: paliers rouges qui entrainent la perte du joueur
